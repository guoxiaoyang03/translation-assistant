# 翻译助手

一个能截图翻译、划词翻译的桌面小工具。

---

## 它能做什么

- 截取屏幕上的文字区域，自动识别并翻译
- 在任何软件里选中文字，按快捷键弹窗翻译
- 可以粘贴文字翻译，也可以导入 txt / docx 文件
- 翻译结果能保存为 txt、Word 或 PDF
- 自动记录翻译历史，可以搜索和回溯
- 最小化窗口后还在后台运行，快捷键随时可用

---

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Shift+Y | 选中文字后按这个，弹窗翻译 |
| Ctrl+Shift+S | 直接启动截图选区 |
